Introduction

Ce cours présente les fondamentaux du fonctionnement d’un site web : les méthodes qui permettent à un site d’être envoyé jusqu’à la machine de l’utilisateur et son affichage (interprétation) dans un navigateur.

Il explique ensuite la séparation du contenu et de la forme, un principe qui sera de nouveau abordé dans les “design patterns” (vu précédemment dans OMB et qui sera développé dans LPB).

Il s’attache à voir extensivement le HTML et le CSS.

Ses objectifs sont de :

  • Connaître les concepts et technologies permettant l’envoi sur le réseau d’un site web et son affichage :
  • L’architecture client-serveur ;
  • Les protocoles de communication réseau ;
  • Les moteurs de rendu d’une page web.
  • Comprendre la séparation entre contenu (HTML) et forme (CSS).
  • Comprendre et programmer en langage HTML.
  • Comprendre et programmer en langage CSS.
  • Connaître les librairies courantes :
    • Exploiter Bootstrap.
    • Exploiter SCSS.
  • Connaître et appliquer les standards du web.
  • Survoler les autres langages de balisage (web).

Évaluations : L’évaluation se fera au moyen d’exercices pratiques journaliers.
